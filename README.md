# PocketFlowFinal

Competition-safe public export of PocketFlow.

This repository intentionally includes:
- Baloss LLM system, model routing, memory and tool scaffolding.
- Archive / Reader app code.
- Builder app.
- MemoPad / Calenotes.
- NewsFlow newsletter builder with public release delivery disabled and no contact lists.
- Notebook/Moltbook agent interface template.
- Screen Relay, Terminal, Web and Router / Baloss Panel surfaces.

This repository intentionally excludes:
- Private discovery, contact, wallet and phone-owner data systems.
- Personal app/account/message/contact seeds.
- Private vault files and generated local records.
- Phone-owner artifacts, generated Android bundles, local downloaded files and private environment files.
- Live newsletter delivery routes and private relay/server endpoint lists.

The complete personal backup should live in the private repository named PocketFlowPrivateTom.
